НАДО СДЕЛАТЬ

Команды:

1. /start - Приветсиве бота, кратко описать для чего нужен бот
2. /about - Рассказать, в чем заключается принцип 1-3-5
3. /help - Под вопросом, если не будет много информации в старт можно написать все команды там
4. /datelist - Ввыбор дня для расписания # База данных
5. /addcase - Добавить дело # База данных
6. /deletecase - Удалить дело # База данных
7. /
